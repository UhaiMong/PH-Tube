Frequently Asked Questions

Discuss the scope of `var`, `let`, and `const`

var has function scope, let has block scope, and const is block-scoped and immutable once declared.

Tell us the use cases of `null` and `undefined`

null is used to represent an intentional absence of a value, while undefined means a variable has been declared but not assigned a value.

What do you mean by REST API?

REST API is an architectural style for designing networked applications using HTTP requests to perform CRUD operations on resources.
